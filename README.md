# SellPlugin 🪙

Minecraft Paper 1.20.x eladási plugin hover tooltipekkel.

## Parancsok
| Parancs | Leírás |
|---------|--------|
| `/sell` | Kézben tartott item eladása |
| `/sell all` | Összes eladása |
| `/sellprice` | Ár lekérdezés |
| `/selladmin set <mat\|hand> <ár>` | Ár beállítás |
| `/selladmin remove <mat\|hand>` | Ár törlés |
| `/selladmin list` | Árlista |
| `/selladmin reload` | Config újratöltés |

## Build
```bash
mvn clean package
```

## Kompatibilitás
- Minecraft 1.20.x
- Paper / Purpur
- Java 17+
